Variables

En JavaScript, las variables son contenedores que almacenan datos o valores, y se definen con las palabras clave var,let y const. La diferencia entre ellas radica en su alcance y si pueden reasignarse o no.

  • var: Declaración de variables que permite reasignación y tiene alcance global o de función (La palabra clave se utilizó en todo el código JavaScript desde 1995 hasta 2015.).
  • let: Permite reasignación, pero tiene un alcance de bloque ({}), lo que la hace más segura en muchos casos.
  • const: Declara constantes, cuyo valor no puede cambiar una vez asignado, y también tiene alcance de bloque.

     // Ejemplo de var
     var nombre = "Juan";
     nombre = "Carlos"; // Reasignación permitida
     console.log(nombre); // Carlos

     // Ejemplo de let
     let edad = 25;
     edad = 30; // Reasignación permitida
     console.log(edad); // 30

     // Ejemplo de const
     const ciudad = "Bogotá";
     // ciudad = "Cali"; // Error: No se puede reasignar una constante
     console.log(ciudad); // Bogotá

Funciones

En JavaScript, las funciones son bloques de código que realizan una tarea específica y pueden ser reutilizadas. Se pueden definir de varias formas y se ejecutan al ser llamadas por su nombre. Las funciones ayudan a organizar y reutilizar el código de manera eficiente.

Tipos de Funciones

  • 1. Función Declarada: Se define con la palabra clave "function" y puede llamarse en cualquier lugar del código. 
    
     function helloWorld() {
       console.log("Hello, World!");
     }
  • 2. Función Expresada: Se almacena en una variable y solo se puede llamar después de su declaración. 
    
     const despedir = function(nombre) {
       return `Adiós, ${nombre}!`;
     };
     console.log(despedir("Juan")); // "Adiós, Juan!"
  • 3. Función Flecha: Introducida en ES6, tiene una sintaxis más corta y no crea su propio contexto this. 
    
     const sumar = (a, b) => a + b;
     console.log(sumar(3, 5)); // 8

Condicionales

En JavaScript, las condicionales se usan para ejecutar diferentes bloques de código según si una condición es verdadera o falsa. Las estructuras condicionales más comunes son if...else, else if y el operador ternario.

Tipos de Condicionales

  • 1. if...else:Ejecuta un bloque de código si una condición es verdadera; de lo contrario, ejecuta otro bloque opcional (else). 
    
     let edad = 18;
     if (edad >= 18) {
       console.log("Eres mayor de edad.");
     } else {
       console.log("Eres menor de edad.");
     }
     // Salida: Eres mayor de edad.
  • 2. else if: Permite agregar múltiples condiciones adicionales. 
    
     let hora = 15;
     if (hora < 12) { 
       console.log("Buenos días"); 
     } else if (hora < 18) { 
       console.log("Buenas tardes"); 
     } else {
       console.log("Buenas noches"); 
     } // Salida: Buenas tardes
  • 3. Operador Ternario: Es una forma simplificada de if...else en una sola línea. 
    
     let esAdulto = edad >= 18 ? "Sí, es adulto" : "No, es menor";
      console.log(esAdulto);
     // Salida: Sí, es adulto

Eventos

En JavaScript, los eventos son acciones que ocurren en el navegador, como un clic de usuario, el envío de un formulario o la carga de una página. Puedes usar eventos para hacer que el sitio sea interactivo y responda a las acciones del usuario. Los eventos permiten que JavaScript reaccione a las interacciones del usuario, mejorando la interactividad en la página web. Se pueden gestionar con addEventListener para cualquier elemento HTML.

Agunos Tipos de Eventos

  • 1. click: Se dispara cuando el usuario hace clic en un elemento. 
    
     const boton = document.getElementById("miBoton");
     boton.addEventListener("click", () => {
       console.log("¡Botón clickeado!");
     });
  • 2. mouseover: Se activa cuando el usuario pasa el ratón sobre un elemento. 
    
     const imagen = document.getElementById("miImagen");
     imagen.addEventListener("mouseover", () => {
       console.log("El ratón está sobre la imagen");
     });
  • 1. load: Se dispara cuando la página web se ha cargado completamente. 
    
     window.addEventListener("load", () => {
       console.log("La página ha cargado completamente.");
     });

Bucles

En JavaScript, los bucles permiten ejecutar un bloque de código repetidamente mientras se cumpla una condición. Son útiles para iterar sobre arrays, objetos o realizar tareas repetitivas. Los bucles más comunes en JavaScript son for, while y do...while.

Tipos de Bucles

  • 1.for: Se usa cuando se sabe cuántas veces se quiere ejecutar el código. 
    
     for (let i = 0; i < 5; i++) { 
       console.log(i); // Imprime los números del 0 al 4 
     }
  • 2.while: Se ejecuta mientras se cumpla una condición. No siempre se conoce cuántas veces se repetirá. 
    
     let i = 0;
     while (i < 5) { 
        console.log(i); // Imprime los números del 0 al 4 i++; 
     }
  • 2.do...while: Similar al while, pero garantiza que el bloque de código se ejecute al menos una vez. 
    
     let i = 0;
     do {
       console.log(i); // Imprime los números del 0 al 4
       i++;
     } while (i < 5);
